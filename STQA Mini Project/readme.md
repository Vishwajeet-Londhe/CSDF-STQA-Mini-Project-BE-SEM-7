## Requirements

### For COVID-19 Website
- XAMPP (Apache + MySQL)
- PHP 7+
- Browser (Chrome/Firefox recommended)
- MySQL Database

### For STQA Test Cases
- Eclipse IDE
- Java JDK 8/11/17+
- TestNG Framework
- Maven (optional)
- Internet connection for dependency download
